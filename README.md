# wsl-dotfiles
My WSL dotfiles

```sh
sh -c "$(curl -fsSL https://raw.githubusercontent.com/alextwothousand/wsl-dotfiles/main/script.sh)"
```
OR
```sh
curl -fsSL https://raw.githubusercontent.com/alextwothousand/wsl-dotfiles/main/script.sh >> script.sh
chmod +x script.sh
nano script.sh
# edit it now...
# ctrl+o, enter, ctrl + x
./script.sh
```
